# Sistema_rh

<h3>Tecnologias utilizadas no projeto:</h3>

<ul>

Para rodar o projeto basta digitar no terminal integrado da pasta:
<pre>

</pre>
Logo em seguida
<pre>
  <span>npm start</span>
</pre>
